TrackMania Competition Patch (Beta)

Published on December 5th, 2020 by donadigo

What is Competition Patch?

TrackMania Competition Patch is a special patch for the game that does not allow means of cheating such as using macros, slowing down the game and other forms of injecting inputs. The patch is in a beta phase, which means there could be bugs or crashes while playing the game.

Why is there a need for this patch?

The patch is meant to solidify & validate records uploaded on important TMX categories such as Official Campaigns and Classic tracks. With the increasing interest in TAS'ing the game, the patch is a leeway into ensuring the leaderboards will stay clean for the forseeable future. None of this has been yet established as a rule in TMX itself, but we're discussing the possibilites of having support implemented for this patch and rules changed in replay uploading (only for important categories that is).

How does it work?

The patch is simply installed into the game's installation directory and from there, the patch will always work when you open the game. Under the hood, the patch overrides the input system of TrackMania so that it can verify that all inputs come from a real hardware device. The patch is in no way invasive - it does not install any low level drivers on your system or collect any data about it.

How do I tell I'm running the game with the patch?

For now the only real way to see, is to check the title of the game's window. If the game is fullscreen, switch to windowed mode and a (Competition Patch) should be visible in the title of the window.

What should I test?

The game should feel just like vanilla in this regard, but the patch adds a new functionality for replay validation. If you've drove a run with the patch, save the replay, edit it and click on the validate button. If you didn't use any tools or macros, the result should say Competition Match after validating. If the result is otherwise, but you still didn't use any tools - then that is a bug in the patch. If the text doesn't show, make sure you are using English language ingame. Replays with different input devices such as gamepad, joystick are appreciated.

Other features

Besides verifying that your replays are valid, the patch also enables you to save replays longer than 2 hours, which is not the case in vanilla game. It also diables restarting the race after clicking on opponents in top left corner, preventing from accidental resets.

Before downloading

This is not a patch in it's final form, bugs or crashes are expected when running this version - you can use it for serious play, however be ready that it might result in a crash or a bug in the patch that makes your replay mismatch. The game should behave just like without it, even though there may be some subtle differences.

How do I install?

  • 1. Navigate to C:\Program Files (x86)\TmUnitedForever or C:\Program Files (x86)\TmNationsForever depending on your game version or to other location if your game is installed elsewhere.
  • 2. Unpack all files to the navigated location. All 4 DLL's should now be extracted in the game's install directory.
  • 3. Done, this process will likely change for the final release.

How do I uninstall?

If you want to uninstall the patch, simply navigate to the game directory again and rename dinput8.dll to anything other than that and the patch should stop working. Deleting files you've unpacked before should also do the job.
